About This Course

COMP 1950 is a required course for the Applied Web Development Associate Certificate and is an elective in Web Application Software Development, as well as the CST Diploma. Students receive hands-on exposure to industry-standard web development methodology and will be able to design and build intermediate level websites. Prerequisites: COMP 1850 or equivalent HTML and CSS knowledge. (3 Credits)

Academic Grading
